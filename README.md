# IISE-Challenge
On IISE, 2026 edition, we worked on a fuel optimization in transport fleets project for Orta Group, more precisely with Orta Transport. We created an AI model that provided the optimized quantity of gas for a driver to reach a given the destination, weather and safety conditions, and their driving style.
